# Actual Budget Monorepo

This repository contains tools and services for [Actual Budget](https://actualbudget.com/).

## Projects

*   **[mcp-server](./mcp-server)**: An MCP (Model Context Protocol) server that exposes your Actual Budget data to LLMs (Claude, ChatGPT, etc.).
*   **[statement-processor](./statement-processor)**: Utilities for processing financial statements.

## Setup

Please refer to the `README.md` in each subdirectory for specific setup instructions.

## Deployment

The `mcp-server` is configured for deployment on Render. See [render.yaml](./render.yaml).
